Security

How we protect your data

Security by Design

AdvisorNote's security architecture is fundamentally different from cloud-based transcription services. By processing everything locally on your device, we eliminate the most significant security risk: transmitting sensitive data over the internet.

100% On-Device Processing

All speech recognition happens entirely on your computer using advanced AI models:

  • • Audio never leaves your device
  • • No internet connection required for transcription
  • • No cloud storage or transmission
  • • Audio files are never saved or logged
  • • Transcriptions remain on your local machine

Zero Data Collection

We cannot access your data because we never collect it:

  • • No telemetry or usage tracking
  • • No analytics or behavioral data
  • • No access to your transcriptions
  • • No access to your audio recordings
  • • No access to client information

Australian Data Sovereignty

Built for Australian financial professionals:

  • • All data processing happens in Australia (on your device)
  • • No data stored on foreign servers
  • • Complies with Australian privacy regulations
  • • No exposure to international data transfer risks

Compliance Ready

AdvisorNote's architecture simplifies compliance:

  • • No third-party data processors to audit
  • • No data breach notification requirements (we don't hold your data)
  • • No cross-border data transfer considerations
  • • Simplified IT security assessments
  • • Easier to satisfy regulatory requirements

Software Updates

We take software security seriously:

  • • Regular security updates and patches
  • • Signed software releases for authenticity
  • • Transparent update process

IT Department Approval

AdvisorNote is designed to pass IT security reviews:

  • • No network traffic containing sensitive data
  • • No external dependencies for core functionality
  • • Minimal attack surface
  • • No cloud infrastructure to secure
  • • Transparent architecture for security audits
